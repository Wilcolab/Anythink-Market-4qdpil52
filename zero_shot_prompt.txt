def to_snake_case(text):
    import re
    text = re.sub(r'[\s\-]+', '_', text)
    text = re.sub(r'([a-z0-9])([A-Z])', r'\1_\2', text)
    return text.lower()

Include error handling for non-string inputs, throw an error with a descriptive message 
if inputs are invalid. Handle edge cases like undefined or null values.
